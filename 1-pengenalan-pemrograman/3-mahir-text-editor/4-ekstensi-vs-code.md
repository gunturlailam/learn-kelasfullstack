# Menggunakan Ekstensi untuk Mempermudah Development

Ekstensi adalah modul tambahan yang bisa diinstal melalui VS Code Marketplace untuk menambah fitur yang tidak ada secara bawaan.

### 🚀 Kenapa Harus Menggunakan Ekstensi?
* **Otomatisasi:** Membantu memformat kode agar rapi secara otomatis.
* **Snippet:** Menyediakan potongan kode siap pakai sehingga tidak perlu mengetik dari nol.
* **Visual:** Memberikan indikasi warna atau ikon yang memudahkan pembacaan struktur folder dan kode.

### 🛠️ Rekomendasi Ekstensi Wajib
1. **Prettier - Code Formatter:** Merapikan kode (indentasi, tanda petik, dll) secara otomatis saat file disimpan.
2. **Bracket Pair Colorizer:** Memberikan warna berbeda pada pasangan kurung ( [ { agar tidak bingung saat koding logika kompleks.`
3. **Auto Close Tag & Auto Rename Tag:** Sangat berguna untuk pengerjaan HTML/Web agar tag otomatis tertutup dan sinkron.
4. **Material Icon Theme:** Mengubah ikon folder dan file agar lebih visual dan mudah dibedakan.
5. **IntelliCode:** Memberikan saran koding berbasis AI yang lebih pintar dari IntelliSense biasa.

### 📦 Ekstensi Spesifik Proyek
* **PHP/Laravel:** PHP Intelephense atau Laravel Blade Snippets.
* **Mobile Development:** Flutter & Dart extensions.

# Menjelajahi Fitur Utama Visual Studio Code

Visual Studio Code (VS Code) bukan sekadar editor teks biasa, melainkan alat pengembangan yang memiliki fitur cerdas untuk mempermudah penulisan kode.

### 🌟 Fitur Unggulan VS Code
1. **IntelliSense:** Fitur yang memberikan saran penulisan kode (autocompletion) berdasarkan variabel, fungsi, dan modul yang digunakan.
2. **Integrated Terminal:** Memungkinkan kita menjalankan command prompt atau git bash langsung di dalam editor tanpa pindah jendela.
3. **Git Integration:** Fitur untuk melakukan commit, pull, dan push ke GitHub secara visual tanpa harus mengetik perintah di terminal.
4. **Extension Marketplace:** Ribuan tambahan fitur (seperti Prettier, ES7+ React/Redux/React-Native snippets, atau Flutter extension) yang bisa diinstal sesuai kebutuhan.
5. **Debugger:** Alat untuk melacak error dan melihat isi variabel secara real-time saat aplikasi dijalankan.

### ⌨️ Shortcut Dasar yang Wajib Diketahui
* **Ctrl + P:** Mencari dan membuka file dengan cepat.
* **Ctrl + Shift + P:** Membuka Command Palette untuk mengakses semua fitur VS Code.
* **Ctrl + \ :** Membuka atau menutup terminal terintegrasi.
* **Alt + Up/Down:** Memindahkan satu baris kode ke atas atau ke bawah.

### 🎨 Kustomisasi
Kita bisa mengubah tema (Color Theme) dan ikon file agar tampilan koding tidak membosankan dan lebih mudah dibaca.
