# Bagaimana Proses Sebuah Program Dibuat?

Untuk menjadi aplikasi yang bisa dijalankan komputer, kode yang ditulis programmer (*Source Code*) harus diproses melalui dua metode utama:

### ⚙️ 1.Compilation (Kompilasi)
Proses di mana kode program diubah secara keseluruhan menjadi bahasa mesin (*Executable Code*) sebelum dijalankan.
* **Contoh Bahasa:** C, C++, C#, Java, Go, Pascal, Swift.

###  ⚡ 2.Interpretation (Interpretasi)
Proses di mana kode program dibaca dan diekseksusi baris demi baris secara langsung menggunakan *Interpreter*.
* **Contoh Bahasa:** JavaScript, PHP, Python, Ruby, Julia.

### 🔄 Alur Proses
1. **Preprocessing:** Tahap persiapan kode awal.
2. **Processing:** Tahap pengubahan kode menjadi instruksi yang dipahami mesin.
