# Mobile Programming

Mobile programming adalah proses pengembangan perangkat lunak untuk berbagai perangkat portabel seperti ponsel, tabel, hingga smartwatch.

### 📱 Sistem Operasi Utama
Dunia mobile saat ini didominasi oleh dua raksasa:
* **Android:** Sistem operasi open-source milik Google.
* **Ios:** Sistem operasi ekslusif milik Apple.

### 🚀 Perkembangan & Teknologi
* **Cross-Platform:** Teknologi masa kini memungkinkan satu basis kode untuk dijalankan di Android dan iOS sekaligus.
* **Ekosistem Luas:** Tidak hanya HP, kini merambah ke *wearables* (smartwatch) dan *Internet of Things* (iOT).
* **Potensi Karier:** Dengan miliaran pengguna aktif smartphone, permintaan akan Mobile Programmer terus meningkat pesat.

> **Info:** Mobile Programming dianggap sebagai sektor teknologi dengan perkembangan paling agresif di masa depan.
