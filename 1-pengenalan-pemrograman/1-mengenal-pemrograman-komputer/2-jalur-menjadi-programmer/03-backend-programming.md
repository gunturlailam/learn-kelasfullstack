# Backend Programming

Backend programming bekerja di "belakang layar" (*server-side*) untuk mengelola data dan logika yang mendukung tampilan frontend.

### 🏗️ Tiga Komponen Utama
1. **Aplikasi:** Logika program yang memproses permintaan.
2. **Server:** Tempat aplikasi berjalan.
3. **Database:** Tempat penyimpanan informasi secara permanen.

### 🛠️ Keahlian yang Dibutuhkan
* **Bahasa Server-Side:** Menguasai bahasa seperti PHP, Python, atau Node.js beserta framework-nya.
* **Pemahaman Frontend:** Dasar HTML, CSS, dan JS agar bisa sinkron dengan tim frontend.
* **Manajemen Data & Server:** Kemampuan administrasi database dan pengaturan server.
* **API Development:** Membangun antarmuka agar aplikasi bisa berkomunikasi antar platform (Web/Mobile).
* **VCS (Git):** Wajib dikuasai untuk kolaborasi tim.

> **Analogi:** Frontend adalah pelayanan dan menu di restoran, sedangkan Backend adalah kaki dan dapur yang memproses pesananmu.
