# Package Manager

Package Manager adalah tools untuk otomatisasi instalasi, konfigurasi, pembaruan, dan penghapusan library atau software secara tuntas.

### 📦 Apa itu Package?
Package berisi file arsip beserta **Metadata** seperti: nama software, versi, vendor, dan daftar **dependensi** (kebutuhan library lain agar program jalan).

### 🚀 Manfaat bagi Programmer
* **Efisiensi:** Tidak perlu menulis kode dari nol jika fitur tersebut sudah tersedia sebagai library.
* **Manajemen Dependensi:** Menangani instalasi library pendukung secara otomatis.
* **Standarisasi:** Memudahkan tim lain menjalankan proyek yang sama dengan library yang versinya identik.

### 🛠️ Contoh Package Manager Populer
* **Composer:** Untuk bahasa PHP.
* **npm:** Untuk Node.js/JavaScript.
* **pip:** Untuk Python.
* **Grade:** Untuk Java/Android.

> **Cara Kerja:** Biasanya mwnggunakan file konfigurasi (seperti composer.json atau package.json) sebagai daftar acuan library yang dibutuhkan proyek.
