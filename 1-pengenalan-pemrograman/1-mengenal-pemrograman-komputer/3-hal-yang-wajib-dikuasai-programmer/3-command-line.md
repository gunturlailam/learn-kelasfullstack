# Command Line Interface (CLI)

CLI adalah antarmuka berbasis teks yang digunakan untuk mengoperasikan sistem atau perangkat lunak melalui perintah ketik.

### ⚔️ CLI vs GLI (Graphical User Interface)
* **GUI:** Menggunakan elemen grafis (ikon, menu) yang ramah bagi pengguna umum.
* **CLI:** Mengandalkan teks. Meski dianggap sulit karena butuh hafalan perintah, CLI menawarkan kecepatan dan efisiensi tinggi.

### 🌟 Mengapa Programmer Memakai CLI?
* **Ringan:** Tanpa beban pemuatan grafis, performa jauh lebih cepat.
* **Otomasi:** Sangat efektif untuk tugas sederhana yang dilakukan berulang kali.
* **Simpel:** Lebih mudah membuat program berbasis teks daripada harus mendesain antarmuka visual.
* **Akses Server:** Kebanyakan server (seperti Linux) dikendalikan melalui CLI.

> **Poin Penting:** Jangan hindari CLI karena terlihat sulit. Semakin akrab kamu dengan terminal, semakin cepat alur kerja pengembangan kodemu.
