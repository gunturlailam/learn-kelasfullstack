# Ringkasan Terminal: Membuat File dengan Perintah `touch`

Dokumen ini merangkum penggunaan perintah `touch` untuk mempercepat alur kerja pengembangan (development) langsung dari terminal Visual Studio Code.

## 1. Apa itu `touch`?
`touch` adalah perintah standar pada terminal (Linux, macOS, dan Git Bash di Windows) yang digunakan untuk:
* **Membuat file baru** yang kosong secara instan.
* **Memperbarui timestamp** (waktu akses/modifikasi) jika file sudah ada tanpa mengubah isinya.

Untuk menggunakan perintah ini di terminal VSC (`Ctrl + ` `), gunakan format berikut:
