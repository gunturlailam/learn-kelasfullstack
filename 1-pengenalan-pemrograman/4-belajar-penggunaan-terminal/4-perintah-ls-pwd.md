# Navigasi Dasar: Mengenal Perintah ls dan pwd

Saat menggunakan terminal, kita perlu mengetahui di mana posisi kita berada dan apa saja isi dari direktori tersebut.

### 📍 1. Perintah pwd (Print Working Directory)
Digunakan untuk menampilkan jalur (path) lengkap dari direktori atau folder tempat kita berada saat ini.
* **Kegunaan:** Memastikan kita berada di folder proyek yang benar sebelum menjalankan perintah seperti git commit atau php artisan.

### 📂 2. Perintah ls (List)
Digunakan untuk menampilkan daftar file dan folder yang ada di dalam direktori saat ini.
* **Variasi Perintah:**
* \ls`: Menampilkan daftar file standar.`
* \ls -a`: Menampilkan semua file, termasuk file tersembunyi (seperti .env atau .git).`
* \ls -l`: Menampilkan daftar dalam format panjang (detail ukuran, izin, dan tanggal).`
* \ls -la`: Kombinasi untuk melihat semua file secara mendetail.`

### 💡 Tips Cepat
Gunakan perintah \clear` jika layar terminal sudah terlalu penuh dengan output perintah sebelumnya agar kembali bersih.`
