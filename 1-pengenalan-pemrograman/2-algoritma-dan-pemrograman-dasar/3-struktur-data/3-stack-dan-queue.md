# Mengenal Struktur Data Stack dan Queue

Stack dan Queue adalah struktur data linier yang mengatur penambahan dan penghapusan data dengan aturan khusus.

### 📚 1. Stack (Tumpukan)
Stack menggunakan prinsip **LIFO** (*Last In, First Out*). Data yang terakhir masuk adalah yang pertama kali keluar.
* **Analogi:** Tumpukan piring atau buku.
* **Operasi Utama:**
* **Push:** Menambah data ke atas tumpukan.
* **Pop:** Mengambil data paling atas.
* **Contoh Penggunaan:** Fitur "Undo/Redo" pada text editor atau tombol "Back" di browser.

### 🚶‍♂️ 2. Queue (Antrean)
Queue menggunakan prinsip **FIFO** (*First In, First Out*). Data yang pertama kali masuk adalah yang pertama kali keluar.
* **Analogi:** Antrean nasabah di bank atau antrean kendaraan di lampu merah.
* **Operasi Utama:**
* **Enqueue:** Menambah data ke belakang antrean.
* **Denqueue:** Mengambil data dari depan antrean.
* **Contoh penggunaan:** Sistem antrean cetak (*print queue*) atau pengiriman pesan di aplikasi chat.
