# Mengenal Struktur Data Binary Tree dan Graph

Struktur data non-linier memungkinkan penyimpanan data dengan hubungan yang lebih kompleks daripada sekedar urutan linier.

### 🌳 1. Binary Tree (Pohon Biner)
Binary tree adalah struktur data hierarkis di mana setiap elemen (node) maksimal memiliki dua "anak" (child), yaitu left child dan right child.
* **Root:** Node paling atas yang menjadi sumber.
* **Leaf:** Node paling bawah yang tidak memiliki anak lagi.
* **Kegunaan:** Sangat efisien untuk proses pencarian data (Binary Search Tree) dan struktur folder di sistem operasi.

### 🕸️ 2. Graph (Graf)
Graph adalah kumpulan simpul (vertex) yang dihubungkan oleh garis (edge) untuk merepresentasikan hubungan antar objek.
* **Vertex:** Titik atau objek data.
* **Edge:** Garis penghubung antar titik.
* **Jenis:** Directed Graph (memiliki arah) dan Undirected Graph (tidak memiliki arah tertentu).
* **Kegunaan:** Digunakan pada sistem peta (Google Maps), rekomendasi pertemanan di media sosial, dan jaringan komputer.
