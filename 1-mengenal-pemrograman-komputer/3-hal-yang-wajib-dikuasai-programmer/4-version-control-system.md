# Version Control System (VSC)

VCS adalah perangkat lunak yang berfungsi mengelola dan merekam setiap perubahan kode program dari waktu ke waktu.

### 🛡️ Mengapa VCS Sangat Penting?
* **Rollback:** Memungkinkan kita kembali ke versi kode sebelumnya jika terjadi kesalahan.
* **Keamanan Aset:** Melindungi source code dari kesalahan manusia atau kehilangan data (sebagai backup).
* **Kolaborasi Tim:** Mencegah terjadinya bentrokan (timpa-menimpa) kode saat bekerja dalam kelompok.
* **Efisiensi:** Membantu programmer bekerja lebih cepat tanpa bingung melacak file mana yang baru diperbarui.

### 🛠️ Mengenal Git
* **Definisi:** Git adalah tools VCS terdistribusi yang paling populer saat ini.
* **Status:** Bersifat *open source* dan gratis.
* **Istilah Lain:** Sering disebut sebagai SCM (*Source Code Management*) atau RCS (*Revision Control System*).

> **Kesimpulan:** Menguasai VCS (khususnya Git) adalah kewajiban bagi setiap programmer modern untuk memastikan alur kerja yang rapi dan aman.
