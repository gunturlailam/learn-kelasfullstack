# Hardware & Embedded System Programming

Hardware programming adalah proses menulis kode tingkat rendah (*low-level*) untuk mengontrol perangkat fisik seperti mikrokontroler, PLC, atau sistem tertanam.

### 🛠️ Bahasa Pemrograman yang Digunakan
* **Low-Level (Assembly):** Digunakan untuk efisiensi maksimal dan kontrol memori yang sangat ketat.
* **High-Level (C/C++):** Bahasa paling populer karena efisien, mudah di*debug*, dan didukung oleh hampir semua produk *embedded*.
* **Modern:** Kini beberapa perangkat sudah mulai mendukung Python dan JavaScript.

### 🧩 Komponen & Pengetahuan Utama
1. **Microcontroller:** Bertindak sebagai CPU perangkat (Contoh: ATMega, ARM, PIC).
2. **Elektronika Dasar:** Pengetahuan wajib tentang sensor, aktuator, dan rangkaian listrik.
3. **Interaksi Fisik:** Program bertujuan untuk mempengaruhi dunia fisik melalui perangkat elektronik.

> **Pesan:** Belajar hardware bukan sekedar menulis kode, tapi memahami bagaimana kode tersebut menggerakkan benda fisik.
