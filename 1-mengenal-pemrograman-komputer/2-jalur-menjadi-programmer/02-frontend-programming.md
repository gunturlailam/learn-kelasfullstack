# Frontend Programming

Frontend programming berfokus pada pembuatan layout website yang interaktif, responsif, dan mudah digunakan oleh pengguna.

### 🛠️ Teknologi Utama Frontend
Apapun yang terlihat di browser adalah kombinasi dari tiga pilar ini:
* **HTML:** Menyusun struktur konten.
* **CSS:** Mengatur gaya,tata letak, dan estetika.
* **JavaScript:** Menambahkan interaktivitas (menu, slider, animasi, dll).

### 🚀 Fokus & Inovasi Modern
* **Framework:** Penggunaan library CSS dan JS untuk mempercepat pengembangan.
* **PWA (Progressive Web App):** Optimasi agar website bisa diakses cepat dan bekerja secara offline.
* **User Experience:** Memastikan navigasi yang informatif dan memikat pengguna.

> **Catatan:** Frontend developer tidak mengurusi infrastruktur atau database, melainkan fokus pada penyajian konten di halaman depan.
