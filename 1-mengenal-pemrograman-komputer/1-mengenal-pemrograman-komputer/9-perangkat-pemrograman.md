# Perangkat yang Dibutuhkan untuk Belajar Pemrograman

Secara garis besar, perangkat utama yang dibutuhkan adalah komputer atau laptop. Spesifikasinya bergantung pada bidang yang ditekuni.

### 💻 Spesifikasi Berdasarkan Bidang
1. **Web Development**
- Cukup RAM 2 GB & Processor Dual Core.
- Butuh aplikasi: Web Server, Code Editor, dan Database Server.
2. **Mobile Development (Android):**
- Minimal RAM 4GB(disarankan lebih agar Android Studio Lancar).
- Butuh smartphone Android fisik untuk testing.
3. **Dekstop & Game Development:**
- Mirip spek Web, namun butuh RAM lebih tinggi dan VGA Card mumpuni untuk Game.
4. **iOS Development:**
- Wajib menggunakan ekosistem Apple (MacBook/iMac).

### 🌐 Kebutuhan Tambahan
* **Layanan Hosting:** Diperlukan saat masuk tahap *development* agar aplikasi bisa diakses via internet.
