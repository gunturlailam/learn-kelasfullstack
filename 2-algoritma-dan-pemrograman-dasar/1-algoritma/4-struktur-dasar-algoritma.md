# Struktur Dasar Algoritma

Terdapat tiga struktur dasar yang menyusun sebuah algoritma pemrograman:

### 1. Struktur Sekuensial (Urutan)
Langkah-langkah diproses secara berurutan mulai dari baris pertama hingga terakhir. Ini adalah cara dasar program dieksekusi.

### 2. Struktur Seleksi (Pemilihan)
Proses akan melalui percabangan berdasarkan kondisi tertentu. Jika kondisi (True), program menjalankan langkah A; jika tidak (False), program menjalnkan langkah B.

### 3. Struktur Perulangan (*Looping*)
Langkah proses akan diulang ke tahap sebelumnya selama kondisi yang diinginkan belum terpenuhi.
* **Contoh:** Menambah nilai X secara terus-menerus sampai mencapai batas yang ditentukan.
