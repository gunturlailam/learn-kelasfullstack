# Mengenal Struktur Data Array (Larik)

Array adalah struktur data yang digunakan untuk menyimpan sekumpulan data dengan tipe yang sama dalam satu variabel tunggal.

### 📦 Karakteristik Utama Array
1. **Homogen:** Setiap elemen dalam satu array harus memiliki tipe data yang sama (misal: semua Integer atau semua String).
2. **Index:** Setiap elemen memiliki nomor urut atau alamat yang disebut index. Di hampir semua bahasa pemrograman, index dimulai dari **0**.
3. **Ukuran Statis:** Dalam algoritma dasar, ukuran array biasanya ditentukan di awal dan tetap.

### 🏗️ Cara Kerja Index
Jika kita punya array NamaSiswa berisi 3 data:
* Index 0: "Andi"
* Index 1: "Budi"
* Index 2: "Caca"
Untuk mengambil data "Budi", kita memanggil NamaSiswa[i].

### 📝 Contoh Pseudocode Deklarasi Array
**DEKLARASI**
ListHarga : ARRAY [1..10] OF Integer
**ALGORITMA**
ListHarga[1] <- 50000
Write(ListHarga[1])
