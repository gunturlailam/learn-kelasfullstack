# Mengenali Kondisi Percabangan (Pemilihan)

Kondisi pemilihan adalah logika di mana program harus memilih satu dari beberapa opsi berdasarkan syarat tertentu untuk melanjutkan proses selanjutnya.

### 👟 Analogi Kasus: Membeli Sepatu
Abdul ingin membeli sepatu, namun anggarannya terbatas. Ia harus memilih antara sepatu A (mahal) atau sepatu B (murah) sesuai usia yang dimilikinya.

### 📊 Visualisasi Flowchart
1. **Mulai:** Pergi ke toko.
2. **Kondisi:** Apakah uang cukup untuk Sepatu A?
3. **Jika YA (True):** Beli Sepatu A.
4. **Jika TIDAK (Fase):** Beli Sepatu B.
5. **Selesai:** Membayar di kasir dan pulang.

### 📝 Penulisan Pseudocode (IF-ELSE)
**DEKLARASI**
budget, harga_A, harga_B : Integer
**ALGORITMA**
IF (harga_A <= budget) THEN
Beli(Sepatu_A)
ELSE
Beli(Sepatu_B)
END IF
