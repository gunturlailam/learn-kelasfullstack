# Apa itu IDE dan Text Editor?

Untuk menulis kode program, kita membutuhkan perangkat lunak khusus. Secara umum, alat ini terbagi menjadi dua kategori: Text Editor dan IDE.

### 📝 1. Text Editor
Program sederhana yang digunakan untuk menulis dan mengedit teks mentah. Untuk keperluan koding, kita menggunakan **Code Editor** yang sudah dilengkapi fitur pendukung.
* **Karakteristik:** Ringan, cepat, dan fiturnya bisa ditambah melalui plugin/ekstensi.
* **Fitur Utama:** Syntax Highlighting (pewarnaan kode) dan Auto-completion.
* **Contoh:** VS Code, Sublime Text, Atom.

### 🏗️ 2. IDE (Integrated Development Environment)
Lingkungan pengembangan yang terintegrasi. IDE menyediakan semua alat yang dibutuhkan programmer dalam satu aplikasi lengkap.
* **Karakteristik:** Berat (memakan banyak RAM), namun sangat powerful karena fiturnya sudah bawaan (*built-in*).
* **Fitur Utama:** Debugger, Compiler/Interpreter, Build Automation, dan Database Manager.
* **Contoh:** Android Studio (untuk Flutter), IntelliJ IDEA, Xcode.

### ⚖️ Perbandingan Singkat
* **Pilih Text Editor (VS Code):** Jika spek laptop terbatas, ingin kecepatan, atau sedang mengerjakan proyek web sederhana.
* **Pilih IDE (Android Studio):** Jika sedang membangun aplikasi kompleks yang butuh analisis kode mendalam dan debugging berat.
