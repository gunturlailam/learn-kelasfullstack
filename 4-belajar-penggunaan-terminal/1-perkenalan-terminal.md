# Kenapa Programmer Perlu Bisa Menggunakan Terminal?

Terminal atau Command Line Interface (CLI) adalah antarmuka berbasis teks yang digunakan untuk memberikan instruksi langsung kepada sistem operasi tanpa melalui antarmuka grafis (GUI).

### 🚀 Alasan Utama Menggunakan Terminal
1. **Efisiensi & Kecepatan:** Menjalankan perintah kompleks sering kali jauh lebih cepat melalui teks daripada mengklik banyak menu di GUI.
2. **Kontrol Penuh:** Banyak fitur sistem operasi dan alat development yang tidak memiliki tampilan grafis.
3. **Otomatisasi:** Memungkinkan pembuatan skrip untuk menjalankan tugas berulang secara otomatis.
4. **Penggunaan Resource Rendah:** Terminal sangat ringan dan tidak membebani RAM, berbeda dengan aplikasi GUI yang berat.

### 🛠️ Peran Terminal dalam Development Modern
* **Version Control:** Menjalankan perintah Git (git commit, git push).
* **Package Manager:** Mengelola dependensi seperti Composer (PHP), NPM (JS), atau Pub (Flutter).
* **Server Management:** Mengelola server cloud (VPS) yang biasanya tidak memiliki tampilan desktop.

### 💻 Istilah Populer
* **CMD / PowerShell:** Terminal bawaan pada sistem operasi Windows.
* **Bash / Zsh:** Terminal standar pada Linux dan macOS.
