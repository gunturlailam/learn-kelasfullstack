# Navigasi Terminal: Berpindah Folder dengan cd

Perintah **cd** (Change Directory) digunakan untuk berpindah dari satu folder ke folder lainnya melalui terminal.Perintah **cd** (Change Directory) digunakan untuk berpindah dari satu folder ke folder lainnya melalui terminal.

### 🚀 Perintah Dasar cd
* **\cd [nama_folder]`**: Masuk ke folder yang ada di dalam direktori saat ini.`
* **\cd ..`**: Keluar satu tingkat ke folder induk (parent directory).`
* **\cd ~`**: Kembali ke folder Home (direktori utama user).`
* **\cd /`**: Berpindah ke direktori root sistem.`
* **\cd -`**: Kembali ke folder sebelumnya tempat kita berada.`

### ⚡ Tips Navigasi Cepat
1. **Autocomplete:** Ketik beberapa huruf awal nama folder lalu tekan tombol **Tab** untuk melengkapi nama folder secara otomatis.
2. **Folder dengan Spasi:** Jika nama folder mengandung spasi, gunakan tanda petik, contoh: \cd "Folder Projek"` atau gunakan backslash: `cd Folder\ Projek`.`
3. **Path Panjang:** Kamu bisa langsung masuk ke folder yang dalam dengan satu perintah, contoh: \cd documents/projek/kickarena`.`
