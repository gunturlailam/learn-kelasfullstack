# Mengenal Flags dan Parameter: Ekstensi Fungsionalitas Perintah CLI

Flags atau Parameters merupakan opsi tambahan yang disertakan setelah perintah utama untuk memodifikasi atau memperluas instruksi standar.

### 🚩 1. Definisi dan Struktur Flags
Flags berfungsi sebagai instruksi spesifik. Biasanya diawali dengan simbol tanda hubung tunggal (-) untuk bentuk singkat atau tanda hubung ganda (--) untuk bentuk panjang.
* Sintaks: command -flag atau command --long-flag

### 🛠️ 2. Implementasi Flags yang Sering Digunakan
Berikut adalah beberapa flags standar dalam navigasi direktori yang relevan dengan alur kerja profesional:
* ls -a: Menampilkan seluruh file, termasuk file tersembunyi (hidden files seperti .env).
* ls -l: Menampilkan daftar direktori dalam format panjang (long format) secara mendetail.
* mkdir -p: Membuat struktur folder bersarang (nested directories) secara otomatis.

### 🧩 3. Penggabungan Beberapa Flags (Chaining)
Untuk efisiensi waktu, terminal memungkinkan penggabungan beberapa flags dalam satu baris perintah:
* Sintaks: ls -la
* Fungsi: Menampilkan seluruh file tersembunyi sekaligus menyajikannya dalam format informasi yang detail.
